    Verb Conjugation Tables¶

    Regular Verb Conjugation¶

    가다 (to go) - Regular Vowel Stem¶

    Form Conjugation Example
    Dictionary 가다
    Stem 가-
    Present (formal) 갑니다 학교에 갑니다
    Present (polite) 가요 학교에 가요
    Present (casual) 가 학교에 가
    Past (formal) 갔습니다 학교에 갔습니다
    Past (polite) 갔어요 학교에 갔어요
    Past (casual) 갔어 학교에 갔어
    Future 갈 거예요 학교에 갈 거예요
    Progressive 가고 있어요 학교에 가고 있어요
    Want to 가고 싶어요 학교에 가고 싶어요
    Can 갈 수 있어요 학교에 갈 수 있어요
    Must 가야 해요 학교에 가야 해요
    Modifier (present) 가는 가는 사람
    Modifier (past) 간 간 사람
    Modifier (future) 갈 갈 사람

    먹다 (to eat) - Regular Consonant Stem¶

    Form Conjugation Example
    Dictionary 먹다
    Stem 먹-
    Present (formal) 먹습니다 밥을 먹습니다
    Present (polite) 먹어요 밥을 먹어요
    Present (casual) 먹어 밥 먹어
    Past (polite) 먹었어요 밥을 먹었어요
    Future 먹을 거예요 밥을 먹을 거예요
    Progressive 먹고 있어요 밥을 먹고 있어요
    Modifier (present) 먹는 먹는 사람
    Modifier (past) 먹은 먹은 음식
    Modifier (future) 먹을 먹을 음식

    크다 (to be big) - Descriptive Verb¶

    Form Conjugation Example
    Dictionary 크다
    Present (formal) 큽니다 집이 큽니다
    Present (polite) 커요 집이 커요
    Past (polite) 컸어요 집이 컸어요
    Modifier (present) 큰 큰 집
    Modifier (past) 컸던/큰 컸던 집

    Note: Descriptive verbs don't use -고 있다 (progressive)

    하다 Verbs¶

    공부하다 (to study)¶

    Form Conjugation
    Present (polite) 공부해요
    Past (polite) 공부했어요
    Future 공부할 거예요
    Progressive 공부하고 있어요
    Want 공부하고 싶어요

    Pattern: All -하다 verbs conjugate the same way - 사랑하다 → 사랑해요 - 노래하다 → 노래해요

    Key Conjugation Patterns¶

    Formal Endings (합니다체)¶

    After vowel: -ㅂ니다 

    가다 → 갑니다
오다 → 옵니다

    After consonant: -습니다 

    먹다 → 먹습니다
읽다 → 읽습니다

    Polite Endings (해요체)¶

    ㅏ, ㅗ stems: -아요 

    가다 → 가 + 아요 → 가요

    Other vowels: -어요 

    먹다 → 먹 + 어요 → 먹어요
서다 → 서 + 어요 → 서요

    하다: 해요 

    하다 → 하 + 여요 → 해요 (contracted)

    Past Tense (-았/었-)¶

    ㅏ, ㅗ stems: -았- 

    가다 → 갔어요
오다 → 왔어요

    Other vowels: -었- 

    먹다 → 먹었어요
서다 → 섰어요

    하다: 했- 

    하다 → 했어요
공부하다 → 공부했어요

    Future Tense (-(으)ㄹ)¶

    After vowel/ㄹ: -ㄹ 

    가다 → 갈 거예요
오다 → 올 거예요
만들다 → 만들 거예요

    After consonant: -을 

    먹다 → 먹을 거예요
읽다 → 읽을 거예요

    Modifiers (Relative Clauses)¶

    Present action verbs: -는 

    가는 사람 (person who goes)
먹는 음식 (food that I eat)

    Descriptive verbs & past: -(으)ㄴ 

    큰 집 (big house)
간 사람 (person who went)
작은 방 (small room)

    Future: -(으)ㄹ 

    갈 곳 (place to go)
먹을 것 (thing to eat)

    Negation¶

    안 (short form): 

    안 가요 (don't go)
안 먹어요 (don't eat)
안 커요 (not big)

    -지 않다 (long form): 

    가지 않아요 (don't go)
먹지 않아요 (don't eat)
크지 않아요 (not big)

    Honorifics (-(으)시-)¶

    After vowel: -시- 

    가다 → 가시다 → 가세요
오다 → 오시다 → 오세요

    After consonant: -으시- 

    먹다 → 먹으시다 → 드세요 (irregular)
읽다 → 읽으시다 → 읽으세요

    Stem Vowel Contractions¶

    Some combinations contract:

    아 + 아 = 아 

    가 + 아요 → 가요 (not 가아요)

    오 + 아 = 와 

    오 + 아요 → 와요

    우 + 어 = 워 

    배우다: 배우 + 어요 → 배워요

    이 + 어 = 여 

    마시다: 마시 + 어요 → 마셔요

    ㅗ + 아 = ㅘ 

    보다: 보 + 아요 → 봐요

    ㅜ + 어 = ㅝ 

    주다: 주 + 어요 → 줘요

    하 + 여 = 해 

    하다: 하 + 여요 → 해요

    Quick Conjugation Steps¶

    1. Identify stem (remove -다)
    2. Check stem ending (vowel or consonant)
    3. Check stem vowel (ㅏ/ㅗ or other)
    4. Apply pattern (합니다체, 해요체, etc.)
    5. Check for contractions/irregulars

    See irregular verbs page for exceptions!
