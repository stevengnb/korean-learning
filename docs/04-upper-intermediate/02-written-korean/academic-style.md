# Academic Writing Style

## Quick Reference

| Form | Usage | Example |
|------|-------|---------|
| **-다 (plain)** | Statement | 연구한다, 중요하다 |
| **-이다** | Noun predicate | 학생이다, 문제이다 |
| **-(으)ㄴ/는** | Modifier | 중요한 문제, 연구하는 방법 |
| **Passive** | Objectivity | 연구되다, 관찰되다 |

## When to Use Academic Style

**Appropriate for:**
- Research papers (논문)
- Academic journals
- Theses and dissertations
- Textbooks
- Scholarly articles
- Scientific reports
- Academic presentations (written)

**Not for:**
- Business letters (use 합니다체)
- Casual writing
- News articles (use modified style)
- General essays (may use 합니다체)

## Formation of Plain Form (-다)

### Action Verbs

**Present tense: -ㄴ다/는다**

**After vowel:**
- Add **-ㄴ다**: 가다 → 간다
- 오다 → 온다

**After consonant:**
- Add **-는다**: 먹다 → 먹는다
- 읽다 → 읽는다

**Irregular ㄹ:**
- ㄹ drops: 만들다 → 만든다
- 살다 → 산다

### Descriptive Verbs

**All descriptive verbs: -다**

- 크다 → 크다 (large)
- 작다 → 작다 (small)
- 좋다 → 좋다 (good)
- 중요하다 → 중요하다 (important)

### 이다 (to be)

**Noun + 이다 / 다**

- 학생이다 (is a student)
- 문제다 (is a problem)
- 연구다 (is research)

### Past Tense

**-았/었다:**

- 갔다 (went)
- 먹었다 (ate)
- 컸다 (was large)
- 중요했다 (was important)

### Future/Presumptive

**-(으)ㄹ 것이다:**

- 갈 것이다 (will go)
- 먹을 것이다 (will eat)
- 클 것이다 (will be large)

## Academic Sentence Patterns

### Making objective statements:

```
이 연구는 한국어 교육의 문제점을 분석한다.
This research analyzes problems in Korean education.

결과는 다음과 같다.
The results are as follows.

이론은 세 가지로 나뉜다.
The theory is divided into three types.
```

### Showing causation:

**-기 때문이다:**
```
이 방법이 효과적이기 때문이다.
This is because this method is effective.
```

**-(으)므로:**
```
중요하므로 자세히 살펴본다.
Since it's important, we examine it in detail.
```

### Showing results:

**-게 되다:**
```
새로운 이론이 발전하게 되었다.
A new theory came to develop.
```

**-ㄴ/는 결과:**
```
연구한 결과 다음과 같은 사실을 알 수 있다.
As a result of research, we can know the following facts.
```

### Citing and referencing:

```
김영수(2020)는 다음과 같이 말한다.
Kim (2020) states as follows.

이에 대해 박민수(2019)는 반대 의견을 제시한다.
Regarding this, Park (2019) presents an opposing view.

선행 연구에 따르면...
According to previous research...
```

## Passive Voice in Academic Writing

### Why use passive?

Academic writing values **objectivity**. Passive voice:
- Focuses on the action, not the actor
- Sounds more objective
- Is standard in scientific writing

### Formation: -되다, -어지다

**-되다 (Sino-Korean verbs):**
```
연구하다 → 연구되다 (to be researched)
관찰하다 → 관찰되다 (to be observed)
분석하다 → 분석되다 (to be analyzed)
발견하다 → 발견되다 (to be discovered)
```

**-어지다 (Native Korean verbs):**
```
만들다 → 만들어지다 (to be made)
알다 → 알려지다 (to be known)
보다 → 보이다 (to be seen)
쓰다 → 쓰이다 (to be used)
```

### Examples in context:

```
이 방법은 많은 연구에서 사용되었다.
This method was used in many studies.

새로운 사실이 발견되었다.
A new fact was discovered.

문제가 해결되지 않았다.
The problem was not solved.

다양한 방법이 시도되었다.
Various methods were attempted.
```

## Sample Academic Paragraph

```
한국어 교육에서 문법 교육의 중요성은 여러 연구에서 강조되어 왔다.
김영수(2020)는 문법이 의사소통 능력의 기초라고 주장한다.
이에 따라 본 연구는 효과적인 문법 교수법을 제시한다.
먼저 기존 연구를 검토한다.
다음으로 새로운 방법을 제안한다.
마지막으로 실험 결과를 분석한다.
연구 결과, 제안한 방법이 효과적임이 밝혀졌다.
따라서 이 방법을 한국어 교육 현장에 적용할 수 있다.
```

**Translation:**
The importance of grammar education in Korean education has been emphasized in various studies. Kim (2020) argues that grammar is the foundation of communication ability. Accordingly, this study presents effective grammar teaching methods. First, we review existing research. Next, we propose a new method. Finally, we analyze experimental results. The research results revealed that the proposed method is effective. Therefore, this method can be applied to Korean education settings.

## Paper Structure

### Typical academic paper sections:

1. **서론 (Introduction)**
   - 연구 배경 (Background)
   - 연구 목적 (Purpose)
   - 연구 문제 (Research questions)

2. **이론적 배경 / 선행 연구 (Literature Review)**
   - 관련 연구 검토
   - 이론적 틀

3. **연구 방법 (Methodology)**
   - 연구 대상 (Subjects)
   - 연구 도구 (Instruments)
   - 연구 절차 (Procedures)

4. **연구 결과 (Results)**
   - 데이터 분석
   - 결과 제시

5. **논의 및 결론 (Discussion & Conclusion)**
   - 결과 해석
   - 시사점 (Implications)
   - 제한점 (Limitations)
   - 후속 연구 제안

### Opening phrases:

```
본 연구의 목적은 ~을 분석하는 것이다.
The purpose of this study is to analyze ~.

이 연구는 ~을 다룬다.
This research deals with ~.

본 논문은 다음과 같이 구성된다.
This paper is structured as follows.
```

### Transition phrases:

```
먼저 / 우선 - First
다음으로 / 그 다음 - Next
마지막으로 - Finally
앞서 언급한 바와 같이 - As mentioned earlier
위에서 살펴본 바와 같이 - As examined above
```

### Concluding phrases:

```
이상에서 살펴본 바와 같이...
As examined above...

결론적으로...
In conclusion...

요약하면...
To summarize...

본 연구의 결과는 다음과 같다.
The results of this study are as follows.
```

## Common Mistakes

**1. Using polite endings:**
```
❌ 이 연구는 중요합니다.
✓ 이 연구는 중요하다.
```

**2. Using casual connectors:**
```
❌ 중요하다. 그래서 연구한다.
✓ 중요하다. 따라서 연구한다.
```

**3. First person overuse:**
```
❌ 내가 연구했다.
✓ 본 연구에서 조사하였다.
```

**4. Inconsistent tense:**
```
❌ 연구했다. 결과는 좋다. 앞으로 연구한다.
✓ 연구하였다. 결과는 좋았다. 앞으로 연구할 것이다.
```

## Sino-Korean vs Native Korean

Academic writing prefers **Sino-Korean** vocabulary:

| Native | Sino-Korean | Meaning |
|--------|-------------|---------|
| 늘다 | 증가하다 | increase |
| 줄다 | 감소하다 | decrease |
| 나누다 | 분류하다 | classify |
| 알아보다 | 조사하다 | investigate |
| 보여주다 | 제시하다 | present |
| 생각 | 견해, 의견 | opinion |
| 때문에 | 이유로 | due to |

## Nominalizations

Convert verbs to nouns for academic tone:

**-기:**
```
연구하다 → 연구하기
분석하다 → 분석하기

한국어를 배우기는 중요하다.
Learning Korean is important.
```

**-음/ㅁ:**
```
알다 → 앎
배우다 → 배움

이러한 앎은 중요하다.
This knowledge is important.
```

**-이:**
```
높다 → 높이 (height, raising)
넓다 → 넓이 (width, area)
```

## Cultural Note

### Academic Culture in Korea

Korean academic writing values:

1. **Objectivity**: Avoid first person, use passive voice
2. **Formality**: Plain form but sophisticated vocabulary
3. **Structure**: Clear organization with sections
4. **Citation**: Proper acknowledgment of sources
5. **Humility**: Acknowledge limitations

### Respect for scholarship:

```
김영수 교수의 연구에 따르면...
According to Professor Kim's research...

선행 연구자들의 업적을 바탕으로...
Based on the achievements of previous researchers...
```

### Common academic phrases showing respect:

```
~의 견해를 존중하면서...
While respecting ~'s view...

~의 중요한 지적처럼...
As in ~'s important observation...
```

## Tips for Academic Writing

1. **Read extensively**: Academic journals in your field
2. **Note patterns**: How do Korean academics structure arguments?
3. **Build vocabulary**: Create lists of field-specific terms
4. **Practice regularly**: Write summaries of articles
5. **Get feedback**: From Korean academics or advanced learners
6. **Use models**: Find well-written papers and analyze structure
7. **Be consistent**: Maintain plain form throughout

Academic Korean takes practice, but mastering it opens doors to Korean scholarship and professional credibility in Korean academic circles.
