# Essay Writing

## Quick Reference

| Section | Purpose | Key Phrases |
|---------|---------|-------------|
| **서론 (Introduction)** | Present topic | 이 글에서는... |
| **본론 (Body)** | Develop argument | 첫째, 둘째, 셋째 |
| **결론 (Conclusion)** | Summarize | 결론적으로... |

## Essay Structure

### Standard 3-Part Structure:

**서론 (Introduction)** - 10-15%
- Hook or background
- Topic introduction
- Thesis statement

**본론 (Body)** - 70-80%
- Main argument 1 + support
- Main argument 2 + support
- Main argument 3 + support
- (Or: Point, Counterpoint, Synthesis)

**결론 (Conclusion)** - 10-15%
- Summarize main points
- Restate thesis
- Final thoughts/implications

## Introduction (서론)

### Opening strategies:

**1. General to specific:**
```
현대 사회에서 기술의 발전은 우리 생활을 크게 변화시켰다.
특히 스마트폰의 보급은...
```

**2. Question:**
```
기술이 과연 우리를 행복하게 만드는가?
이 질문에 답하기 위해...
```

**3. Statistics or facts:**
```
최근 조사에 따르면 한국인의 90%가 스마트폰을 사용한다.
이러한 변화는...
```

**4. Quote or saying:**
```
"기술은 양날의 검이다"는 말이 있다.
이는...
```

### Thesis statement patterns:

```
이 글에서는 ~을/를 논의하고자 한다.
In this essay, I will discuss ~.

본고는 ~을/를 주장한다.
This paper argues that ~.

~에 대해 살펴보고자 한다.
I will examine ~.

~의 문제점과 해결 방안을 제시하고자 한다.
I will present problems and solutions regarding ~.
```

**Examples:**
```
이 글에서는 스마트폰 사용의 장단점을 논의하고자 한다.

본고는 환경 보호가 시급한 과제임을 주장한다.

한국어 교육의 현황과 개선 방안을 살펴보고자 한다.
```

## Body (본론)

### Organizing arguments:

**Method 1: Listing (열거)**

```
첫째, ...
둘째, ...
셋째, ...
마지막으로, ...
```

**Method 2: Pros and Cons (장단점)**

```
먼저 장점을 살펴보면, ...
반면 단점은 ...
```

**Method 3: Cause and Effect (원인과 결과)**

```
이러한 문제의 원인은 ...
그 결과 ...
```

**Method 4: Problem and Solution (문제와 해결책)**

```
첫 번째 문제는 ...
이를 해결하기 위해 ...
```

### Paragraph structure (PEEL):

**P**oint - Main idea
**E**xplanation - Explain the point
**E**xample - Give evidence
**L**ink - Connect to thesis

**Example paragraph:**
```
[Point] 첫째, 스마트폰은 의사소통을 편리하게 만든다.

[Explanation] 과거에는 전화나 편지를 통해서만 연락할 수 있었지만,
이제는 문자, 영상통화 등 다양한 방법으로 소통할 수 있다.

[Example] 예를 들어, 해외에 있는 가족과도 실시간으로
얼굴을 보며 대화할 수 있다.

[Link] 따라서 스마트폰은 현대인의 필수품이라고 할 수 있다.
```

### Transition words and phrases:

**Ordering:**
```
먼저 / 우선 - first
다음으로 / 그 다음 - next
또한 / 게다가 - also, moreover
마지막으로 - finally
```

**Adding information:**
```
뿐만 아니라 - not only... but also
더욱이 - furthermore
더 나아가 - moreover
특히 - especially
실제로 - in fact
```

**Contrasting:**
```
그러나 - however
반면 - on the other hand
하지만 - but
물론 - of course
그럼에도 불구하고 - nevertheless
```

**Showing cause/effect:**
```
따라서 - therefore
그러므로 - thus
그 결과 - as a result
왜냐하면 - because
~(으)므로 - since
~기 때문에 - because
```

**Giving examples:**
```
예를 들어 - for example
가령 - for instance
즉 - that is
구체적으로 말하면 - specifically
```

## Conclusion (결론)

### Concluding strategies:

**1. Summarize main points:**
```
이상에서 살펴본 바와 같이...
As examined above...

앞서 논의한 내용을 요약하면...
To summarize what was discussed...
```

**2. Restate thesis:**
```
결론적으로, ~는/은 ...이다.
In conclusion, ~ is...

따라서 ~해야 한다.
Therefore, we must...
```

**3. Call to action:**
```
우리는 ~해야 할 것이다.
We should...

이를 위해 노력이 필요하다.
Efforts are needed for this.
```

**4. Future implications:**
```
앞으로 ~할 것으로 기대된다.
It is expected that ~ in the future.

~가 중요한 과제로 남아 있다.
~ remains an important task.
```

### Complete conclusion example:

```
결론적으로, 스마트폰은 현대 사회에서 중요한 역할을 한다.
의사소통, 정보 검색, 업무 처리 등 다양한 면에서 우리 생활을 편리하게 만들었다.
물론 과도한 사용은 문제가 될 수 있지만,
적절히 활용한다면 스마트폰은 유용한 도구가 될 것이다.
따라서 우리는 스마트폰의 장점을 살리면서
단점을 최소화하는 지혜가 필요하다.
```

**1. Social issues:**

## Useful Essay Expressions

### Presenting your opinion:

```
~라고 생각한다 / ~라고 본다
I think that...

~가 필요하다고 생각한다
I think ~ is necessary

개인적으로 ~라고 생각한다
Personally, I think...

~는 것이 바람직하다
It is desirable that...
```

### Showing agreement/disagreement:

```
~에 동의한다
I agree with ~

~는 타당하다
~ is valid

그러나 ~는 문제가 있다
However, ~ is problematic

~에는 반대한다
I oppose ~
```

### Emphasizing:

```
특히 - especially
무엇보다 - above all
가장 중요한 것은 - most importantly
주목해야 할 점은 - the point to note is
강조하고 싶은 것은 - what I want to emphasize is
```

### Acknowledging other views:

```
물론 ~라는 의견도 있다
Of course, there is also the opinion that...

~라고 주장하는 사람들도 있다
There are people who argue that...

반대 의견도 존재한다
Opposing views also exist

일리가 있지만 - it has merit, but...
```

## Sample Essay (TOPIK Style)

**주제: 외국어를 배우는 것이 왜 중요한가?**

```
[서론]
세계화 시대에 외국어 능력은 점점 더 중요해지고 있다.
특히 영어를 비롯한 여러 외국어를 구사할 수 있는 능력은
개인의 경쟁력을 높이는 중요한 요소가 되었다.
이 글에서는 외국어 학습이 중요한 이유를 세 가지 측면에서 살펴보고자 한다.

[본론 1]
첫째, 외국어는 더 많은 기회를 제공한다.
외국어를 잘하면 취업 기회가 늘어나고 국제적인 무대에서 활동할 수 있다.
예를 들어, 글로벌 기업에서 일하거나 해외에서 공부할 수 있는
기회를 얻을 수 있다.
따라서 외국어 능력은 개인의 성공에 중요한 역할을 한다.

[본론 2]
둘째, 외국어 학습은 다른 문화를 이해하는 데 도움이 된다.
언어는 단순한 의사소통 도구가 아니라 그 나라의 문화와 사고방식을 담고 있다.
외국어를 배우면서 다른 문화를 접하고 이해하게 되면
세계를 보는 시야가 넓어진다.
이는 국제화 시대에 꼭 필요한 능력이라고 할 수 있다.

[본론 3]
셋째, 외국어 학습은 두뇌 발달에도 긍정적인 영향을 미친다.
연구에 따르면 여러 언어를 사용하는 사람들은
인지 능력과 문제 해결 능력이 더 뛰어나다고 한다.
또한 치매 예방에도 효과가 있다는 연구 결과도 있다.
즉, 외국어 학습은 건강한 두뇌를 유지하는 데도 도움이 된다.

[결론]
이상에서 살펴본 바와 같이 외국어 학습은 다양한 측면에서 중요하다.
취업과 진로, 문화 이해, 그리고 두뇌 건강까지
외국어는 우리에게 많은 이점을 제공한다.
물론 외국어를 배우는 것이 쉽지는 않지만,
꾸준히 노력한다면 누구나 외국어를 습득할 수 있다.
따라서 우리는 외국어 학습의 중요성을 인식하고
적극적으로 공부해야 할 것이다.
```

## Common Mistakes

**1. No clear thesis:**
```
❌ 스마트폰에 대해 이야기하겠다.
✓ 이 글에서는 스마트폰의 장단점을 분석하고자 한다.
```

**2. Weak topic sentences:**
```
❌ 다음은 두 번째 이유다.
✓ 둘째, 외국어 학습은 문화 이해에 도움이 된다.
```

**3. Lack of examples:**
```
❌ 운동은 건강에 좋다. (no support)
✓ 운동은 건강에 좋다. 예를 들어, 규칙적인 운동은 심장 질환을 예방한다.
```

**4. Abrupt conclusion:**
```
❌ 그래서 외국어가 중요하다. (too sudden)
✓ 이상에서 살펴본 바와 같이 외국어는 여러 측면에서 중요하다.
```

**5. Mixing formality:**
```
❌ 외국어는 중요하다. 그래서 공부해야 돼.
✓ 외국어는 중요하다. 따라서 공부해야 한다.
```

## Writing Tips

### Planning (10 minutes):

1. **Understand the prompt** - What is being asked?
2. **Brainstorm** - List main ideas
3. **Organize** - Choose 2-3 main points
4. **Outline** - Structure introduction, body, conclusion

### Writing (30-35 minutes):

1. **Introduction** - Hook, background, thesis
2. **Body** - One paragraph per main point with examples
3. **Conclusion** - Summarize and final thoughts

### Checking (5-10 minutes):

1. **Content** - Does it answer the question?
2. **Structure** - Clear introduction, body, conclusion?
3. **Grammar** - Check verb endings, particles
4. **Vocabulary** - Appropriate level and variety?
5. **Length** - Meeting requirements?

## Essay Formality Level

For TOPIK and academic essays, use:

**Ending style:** Mainly -다 (plain form) or -ㅂ니다/습니다 (formal)

**Most common for TOPIK:** Mix of both
- Academic tone: -다
- Polite academic: -ㅂ니다/습니다

**Be consistent** within each paragraph.

**Avoid:**
- -아/어요 (too casual for essays)
- -네요, -군요 (too conversational)
- Contractions (뭐, 거, etc.)
- Slang or very casual expressions

## Cultural Note

### Korean Essay Expectations

Korean essays value:

1. **Clear structure** - Introduction, body, conclusion must be obvious
2. **Balance** - Present multiple perspectives
3. **Humility** - Use 생각한다 rather than stating as absolute fact
4. **Examples** - Support all claims with evidence
5. **Cohesion** - Smooth transitions between ideas

### Common essay lengths:

- **TOPIK II**: 200-300 characters (Task 53), 600-700 characters (Task 54)
- **University essays**: 2-3 pages (A4)
- **Graduate papers**: 5-10+ pages

Practice regularly to improve your essay writing skills. Read model essays, analyze their structure, and write frequently to develop your own style while maintaining proper Korean essay conventions.
