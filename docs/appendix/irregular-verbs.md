# Irregular Verbs

## ㄷ Irregular

**When ㄷ final consonant meets vowel, ㄷ becomes ㄹ**

### Examples:
- 듣다 (to hear) → 들어요, 들었어요
- 걷다 (to walk) → 걸어요, 걸었어요
- 묻다 (to ask) → 물어요, 물었어요

**Conjugation:**
```
듣다:
Present: 들어요 (not 듣어요)
Past: 들었어요
Modifier: 듣는 (ㄷ stays before -는)
```

**Exception:** 받다 (receive) is regular → 받아요

## ㅂ Irregular

**When ㅂ final meets vowel, ㅂ becomes 우**

### Examples:
- 춥다 (cold) → 추워요
- 덥다 (hot) → 더워요
- 돕다 (help) → 도와요
- 아름답다 (beautiful) → 아름다워요

**Conjugation:**
```
춥다:
Present: 추워요 (not 춥어요)
Past: 추웠어요
Modifier: 추운
```

**Exception:** 좁다 (narrow) can be regular → 좁아요

## ㅅ Irregular

**When ㅅ final meets vowel, ㅅ disappears**

### Examples:
- 짓다 (build) → 지어요
- 낫다 (better) → 나아요
- 잇다 (connect) → 이어요

**Conjugation:**
```
짓다:
Present: 지어요 (not 짓어요)
Past: 지었어요
Modifier: 짓는 (ㅅ stays before consonants)
```

## ㅎ Irregular

**When ㅎ meets ㄴ or vowel, ㅎ disappears**

### Examples:
- 빨갛다 (red) → 빨개요, 빨간
- 노랗다 (yellow) → 노래요, 노란
- 하얗다 (white) → 하얘요, 하얀

**Conjugation:**
```
빨갛다:
Present: 빨개요 (not 빨갛어요)
Modifier: 빨간 (not 빨갛은)
Past: 빨갰어요
```

## ㄹ Irregular

**ㄹ drops before ㄴ, ㅂ, ㅅ**

### Examples:
- 살다 (live) → 살아요, BUT: 삽니다, 사는
- 만들다 (make) → 만들어요, BUT: 만듭니다, 만드는
- 알다 (know) → 알아요, BUT: 압니다, 아는

**Conjugation:**
```
살다:
Present polite: 살아요 (keep ㄹ)
Present formal: 삽니다 (ㄹ drops before ㅂ)
Modifier: 사는 (ㄹ drops before ㄴ)
Past: 살았어요
```

## 르 Irregular

**르 + vowel → double ㄹ + 라/러**

### Examples:
- 모르다 (not know) → 몰라요
- 부르다 (call, sing) → 불러요
- 빠르다 (fast) → 빨라요

**Conjugation:**
```
모르다:
Present: 몰라요 (모르 + 아 → 몰 + 라)
Past: 몰랐어요
Modifier: 모르는
```

## 으 Irregular

**으 drops before -아/어**

### Examples:
- 쓰다 (write, use) → 써요
- 끄다 (turn off) → 꺼요
- 크다 (big) → 커요

**Conjugation:**
```
쓰다:
Present: 써요 (쓰 + 어 → 써)
Past: 썼어요
Modifier: 쓰는 (으 stays before consonants)
```

**Exception:** 따르다 (follow) is regular → 따라요

## Quick Test

Are these irregular?

1. 받다 (receive) - No (regular)
2. 굽다 (grill) - Yes (ㅂ irregular) → 구워요
3. 걷다 (walk) - Yes (ㄷ irregular) → 걸어요
4. 걷다 (roll up) - No (regular) → 걷어요
5. 이르다 (early) - Yes (르 irregular) → 일러요

**Note:** Some verbs can be both regular and irregular with different meanings!

## Common Irregular Verbs to Memorize

**ㄷ Irregular:**
듣다, 걷다, 묻다, 싣다

**ㅂ Irregular:**
춥다, 덥다, 춥다, 돕다, 아름답다, 가볍다

**ㅅ Irregular:**
짓다, 낫다

**ㅎ Irregular:**
빨갛다, 노랗다, 하얗다, 파랗다, 까맣다

**ㄹ Irregular:**
살다, 만들다, 알다, 팔다

**르 Irregular:**
모르다, 부르다, 빠르다, 고르다

**으 Irregular:**
쓰다, 끄다, 크다, 바쁘다

Practice these until conjugation becomes automatic!
