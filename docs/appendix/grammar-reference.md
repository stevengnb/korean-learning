# Grammar Reference - Quick Lookup

## Sentence Endings by Formality

| Formality | Statement | Question | Proposal | Command |
|-----------|-----------|----------|----------|---------|
| **Formal (합니다체)** | -습니다/-ㅂ니다 | -습니까/-ㅂ니까 | -(으)ㅂ시다 | -(으)십시오 |
| **Polite (해요체)** | -아/어요 | -아/어요? | -(으)ㄹ까요? | -(으)세요 |
| **Casual (반말)** | -아/어 | -아/어? | -(으)ㄹ까? | -아/어 |

## Past Tense

**-았/었/했-**
- After ㅏ, ㅗ: -았-
- After other vowels: -었-
- 하다 → 했-

```
가다 → 갔다
먹다 → 먹었다
하다 → 했다
```

## Future Tense

**-(으)ㄹ 거예요** - will (casual prediction)
**-(으)ㄹ게요** - will (speaker's intention)
**-(으)ㄹ 것이다** - will (formal/written)

## Connectors

| Pattern | Meaning | Example |
|---------|---------|---------|
| **-고** | And (sequential) | 밥을 먹고 갔어요 |
| **-아/어서** | So, because | 바빠서 못 갔어요 |
| **-(으)니까** | Since, because | 비가 오니까 우산을 가져가세요 |
| **-지만** | But, although | 비싸지만 샀어요 |
| **-(으)면** | If, when | 시간이 있으면 올게요 |
| **-(으)면서** | While | 음악을 들으면서 공부해요 |

## Modifiers

**Action verbs (present):** -는
```
가는 사람 - person who goes
먹는 음식 - food (that I) eat
```

**Descriptive verbs & past:** -(으)ㄴ
```
큰 집 - big house
간 사람 - person who went
```

**Future:** -(으)ㄹ
```
갈 사람 - person who will go
먹을 음식 - food (I'll) eat
```

## Ability & Possibility

**-(으)ㄹ 수 있다/없다** - can/cannot
```
한국어를 할 수 있어요 - can speak Korean
갈 수 없어요 - cannot go
```

## Intention & Planning

**-(으)려고 하다** - intend to
**-고 싶다** - want to
**-(으)ㄹ까 하다** - thinking about doing
**-기로 하다** - decided to

## Experience

**-아/어 본 적이 있다/없다** - have/haven't done
```
한국에 가 본 적이 있어요 - I've been to Korea
먹어 본 적이 없어요 - I've never eaten it
```

## Obligation

**-아/어야 하다/되다** - must, have to
**-(으)면 안 되다** - must not

## Permission

**-아/어도 되다** - may, it's okay to
**-(으)면 안 되다** - must not

## Trying & Attempting

**-아/어 보다** - try doing
```
먹어 봐요 - Try eating it
```

## Giving & Receiving

**주다** - give (to someone)
**드리다** - give (humble)
**받다** - receive
**-아/어 주다** - do for someone

## Honorifics

**Subject:** -(으)시-
**Special verbs:**
- 먹다 → 드시다/잡수시다
- 자다 → 주무시다
- 있다 → 계시다
- 말하다 → 말씀하시다

## Quick Grammar Patterns

**-고 있다** - be doing (progressive)
**-아/어 있다** - be in a state
**-게 되다** - come to be
**-아/어지다** - become
**-(으)ㄴ/는데** - but, and (background)
**-거든요** - you know, because
**-잖아요** - you know (as you know)
**-(이)라서** - because (it's)
