# Questions

!!! abstract "Quick Reference"
    | Type | Pattern | Shortened | Example |
    |------|---------|-----------|---------|
    | **Verb (Present)** | -냐고/느냐고 묻다 | -냐고/느냐고 | 가냐고 물었어요 |
    | **Verb (Past)** | -았/었냐고 묻다 | -았/었냐고 | 갔냐고 물었어요 |
    | **Adjective** | -(으)냐고 묻다 | -(으)냐고 | 좋으냐고 물었어요 |
    | **Noun (이다)** | -(이)냐고 묻다 | -(이)냐고 | 학생이냐고 물었어요 |
    | **All types** | | -래 (casual) | 가냐고 → 가래 |

    **Key Point**: Use 묻다/물어보다 (to ask) instead of 하다; -래 can mean question or command!

## Detailed Explanation

Reported questions convey questions that someone asked. Unlike statements, questions typically use 묻다 (to ask) as the reporting verb, though 하다 can also be used.

### Verb Questions

**Present Tense:**

Formation differs by verb ending:

- **Verbs ending in consonant**: -느냐고 묻다
  - 먹다 → 먹느냐고 물었어요
  - 읽다 → 읽느냐고 물었어요

- **Verbs ending in vowel/ㄹ**: -냐고 묻다
  - 가다 → 가냐고 물었어요
  - 살다 → 사냐고 물었어요

- **하다 verbs**: -하냐고 묻다
  - 공부하다 → 공부하냐고 물었어요

**Past Tense:**

All verbs: -았/었냐고 묻다

- 먹다 → 먹었냐고 물었어요
- 가다 → 갔냐고 물었어요
- 공부하다 → 공부했냐고 물었어요

**Future Tense:**

-(으)ㄹ 거냐고 묻다

- 먹다 → 먹을 거냐고 물었어요
- 가다 → 갈 거냐고 물었어요

### Adjective Questions

**Present Tense:**

Add -(으)냐고 to the stem:

- **After vowel or ㄹ**: -냐고
  - 크다 → 크냐고 물었어요
  - 어떻다 → 어떠냐고 물었어요

- **After consonant**: -으냐고
  - 좋다 → 좋으냐고 물었어요
  - 맛있다 → 맛있냐고 물었어요

**Past Tense:**

-았/었냐고 묻다

- 좋다 → 좋았냐고 물었어요
- 예쁘다 → 예뻤냐고 물었어요

### Noun Questions (이다)

**After consonant**: -이냐고 묻다

- 학생이다 → 학생이냐고 물었어요
- 의사이다 → 의사냐고 물었어요

**After vowel**: -냐고 묻다

- 선생님이다 → 선생님이냐고 물었어요
- 가수이다 → 가수냐고 물었어요

**Past**: -이었냐고/였냐고 묻다

- 학생이었다 → 학생이었냐고 물었어요

### Wh-Questions

For questions with question words (뭐, 어디, 누구, 언제, etc.), use the same patterns:

- 뭐 먹느냐고 물었어요 (asked what I'm eating)
- 어디 가냐고 물었어요 (asked where I'm going)
- 누구냐고 물었어요 (asked who it is)
- 언제 오냐고 물었어요 (asked when I'm coming)
- 왜 안 먹느냐고 물었어요 (asked why I'm not eating)

### Formation Steps

1. Take the direct question
2. Remove question ending (요?/니?/냐?)
3. Identify verb/adjective/noun type
4. Add appropriate -냐고/느냐고 ending
5. Add 묻다/물어보다 (conjugate as needed)
6. Optional: Use shortened form

### Examples

**Present Tense Verbs:**

Direct → Reported

- "한국어를 공부해요?" → 한국어를 공부하냐고 물었어요
- "어디 가요?" → 어디 가냐고 물었어요
- "밥을 먹어요?" → 밥을 먹느냐고 물었어요
- "책을 읽어요?" → 책을 읽느냐고 물었어요
- "운동해요?" → 운동하냐고 물었어요

**Past Tense Verbs:**

- "영화를 봤어요?" → 영화를 봤냐고 물었어요
- "어디 갔어요?" → 어디 갔냐고 물었어요
- "점심 먹었어요?" → 점심 먹었냐고 물었어요
- "숙제를 했어요?" → 숙제를 했냐고 물었어요

**Future Tense:**

- "내일 올 거예요?" → 내일 올 거냐고 물었어요
- "같이 갈 거예요?" → 같이 갈 거냐고 물었어요

**Adjectives:**

- "한국어가 어려워요?" → 한국어가 어렵냐고 물었어요
- "날씨가 좋아요?" → 날씨가 좋으냐고 물었어요
- "음식이 맛있어요?" → 음식이 맛있냐고 물었어요
- "방이 깨끗해요?" → 방이 깨끗하냐고 물었어요
- "커요?" → 크냐고 물었어요

**Nouns (이다):**

- "학생이에요?" → 학생이냐고 물었어요
- "의사예요?" → 의사냐고 물었어요
- "한국 사람이에요?" → 한국 사람이냐고 물었어요
- "선생님이에요?" → 선생님이냐고 물었어요

**Wh-Questions:**

- "뭐 해요?" → 뭐 하냐고 물었어요
- "누구예요?" → 누구냐고 물었어요
- "어디 살아요?" → 어디 사냐고 물었어요
- "왜 안 왔어요?" → 왜 안 왔냐고 물었어요
- "언제 만날까요?" → 언제 만나냐고 물었어요

**Complex Sentences:**

- 친구가 제가 어디 사냐고 물어봤어요 (My friend asked where I live)
- 엄마가 밥을 먹었냐고 물으셨어요 (Mom asked if I ate)
- 선생님이 숙제를 했냐고 물으셨어요 (The teacher asked if I did homework)
- 오빠가 뭐 하고 있냐고 물어봤어요 (My brother asked what I'm doing)

### Alternative Reporting Verbs

**물어보다 (to ask - casual):**

- 가냐고 물어봤어요 (asked if going)

**질문하다 (to question - formal):**

- 학생이냐고 질문했어요 (questioned if student)

**하다 (to say - also acceptable):**

- 좋으냐고 했어요 (said/asked if it's good)

### Shortened Form -래

In very casual speech, -냐고 can become -래:

- 어디 가냐고 → 어디 가래 (where are you going, they ask)
- 뭐 하냐고 → 뭐 하래 (what are you doing, they ask)

**Warning**: -래 is ambiguous and can mean:
1. Reported question: 뭐 하래? (What are they asking you to do?)
2. Reported command: 하래 (They told me to do it)

Context determines meaning!

### Tense in Reporting Verb

묻다/물어보다 is usually in past tense (물었어요/물어봤어요) because you're reporting a question that was already asked:

- 가냐고 물어요 (asks if I'm going - habitual)
- 가냐고 물었어요 (asked if I'm going - one time)
- 가냐고 물어볼 거예요 (will ask if I'm going - future)

### Yes/No vs Wh-Questions

Both use the same pattern in Korean:

- Yes/No: 좋아하냐고 물었어요 (asked if I like it)
- Wh-Question: 뭐 좋아하냐고 물었어요 (asked what I like)
