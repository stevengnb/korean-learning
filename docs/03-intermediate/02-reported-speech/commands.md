# Commands

!!! abstract "Quick Reference"
    | Type | Pattern | Shortened | Example |
    |------|---------|-----------|---------|
    | **Positive Command** | -(으)라고 하다 | -래(요) | 가라고 했어요 → 가래요 |
    | **Negative Command** | -지 말라고 하다 | -지 말래(요) | 먹지 말라고 했어요 |
    | **Polite Request** | -(으)시라고 하다 | -(으)시래요 | 오시라고 했어요 |
    | **Suggestion** | -자고 하다 | -재(요) | 가자고 했어요 → 가재요 |

    **Key Point**: -(으)라고 for commands; -지 말라고 for negative; -자고 for suggestions

## Detailed Explanation

Reported commands convey orders, requests, or suggestions that someone gave. The pattern differs from statements and questions.

### Positive Commands -(으)라고 하다

**Formation**:

- **After vowel or ㄹ**: -라고 하다
  - 가다 → 가라고 하다 → 가래요
  - 살다 → 살라고 하다 → 살래요

- **After consonant**: -으라고 하다
  - 먹다 → 먹으라고 하다 → 먹으래요
  - 읽다 → 읽으라고 하다 → 읽으래요

- **하다 verbs**: -하라고 하다
  - 공부하다 → 공부하라고 하다 → 공부하래요

**Key Features:**

- Direct commands or orders
- Can sound quite strong or blunt
- Often from someone in authority
- Use 시 for polite commands to superiors

### Negative Commands -지 말라고 하다

**Formation**: Verb stem + -지 말라고 하다

- 가다 → 가지 말라고 하다 → 가지 말래요
- 먹다 → 먹지 말라고 하다 → 먹지 말래요
- 하다 → 하지 말라고 하다 → 하지 말래요

**Key Features:**

- Tell someone NOT to do something
- Same pattern for all verbs (no vowel/consonant distinction)
- Can be softened with 달라고 instead of 말라고

### Polite Commands -(으)시라고 하다

**Formation**: Add -시- for honorific requests:

- 가다 → 가시라고 하다
- 먹다 → 드시라고 하다 (special honorific)
- 오다 → 오시라고 하다

### Suggestions -자고 하다

**Formation**: Verb stem + -자고 하다

- 가다 → 가자고 하다 → 가재요
- 먹다 → 먹자고 하다 → 먹재요
- 공부하다 → 공부하자고 하다 → 공부하재요

**Key Features:**

- "Let's..." suggestions
- Inclusive (includes speaker)
- Softer than commands
- Common in casual speech

### Formation Steps

**For positive commands:**

1. Take the verb stem
2. Check if ends in vowel/ㄹ or consonant
3. Add -라고 or -으라고
4. Add 하다 (conjugate as needed)
5. Optional: Use -래요 shortening

**For negative commands:**

1. Take the verb stem
2. Add -지 말라고
3. Add 하다 (conjugate as needed)
4. Optional: Use -지 말래요

**For suggestions:**

1. Take the verb stem
2. Add -자고
3. Add 하다 (conjugate as needed)
4. Optional: Use -재요

### Examples

**Positive Commands:**

Direct → Reported (Formal) → Shortened

- "앉으세요" → 앉으라고 했어요 → 앉으래요
- "가세요" → 가라고 했어요 → 가래요
- "먹어요" → 먹으라고 했어요 → 먹으래요
- "공부하세요" → 공부하라고 했어요 → 공부하래요
- "기다리세요" → 기다리라고 했어요 → 기다리래요
- "읽으세요" → 읽으라고 했어요 → 읽으래요

**Negative Commands:**

- "가지 마세요" → 가지 말라고 했어요 → 가지 말래요
- "먹지 마세요" → 먹지 말라고 했어요 → 먹지 말래요
- "걱정하지 마세요" → 걱정하지 말라고 했어요 → 걱정하지 말래요
- "늦지 마세요" → 늦지 말라고 했어요 → 늦지 말래요
- "잊지 마세요" → 잊지 말라고 했어요 → 잊지 말래요

**Polite Requests (with 시):**

- "앉으세요" (to elder) → 앉으시라고 했어요
- "오세요" → 오시라고 했어요
- "드세요" → 드시라고 했어요
- "쉬세요" → 쉬시라고 했어요

**Suggestions:**

- "같이 가요" → 같이 가자고 했어요 → 같이 가재요
- "영화 봐요" → 영화 보자고 했어요 → 영화 보재요
- "밥 먹어요" → 밥 먹자고 했어요 → 밥 먹재요
- "내일 만나요" → 내일 만나자고 했어요 → 내일 만나재요
- "공부해요" → 공부하자고 했어요 → 공부하재요

**Complex Sentences:**

- 선생님이 숙제를 하라고 하셨어요 (The teacher told me to do homework)
- 엄마가 야채를 먹으라고 했어요 (Mom told me to eat vegetables)
- 친구가 늦지 말라고 했어요 (My friend told me not to be late)
- 오빠가 같이 가자고 했어요 (My brother suggested we go together)
- 의사가 담배를 피우지 말라고 했어요 (The doctor told me not to smoke)

### Different Reporting Verbs

**시키다 (to order, make someone do):**

- 청소하라고 시켰어요 (ordered me to clean)
- 일찍 오라고 시켰어요 (ordered me to come early)

**부탁하다 (to request, ask a favor):**

- 도와달라고 부탁했어요 (asked me to help)
- 전화하라고 부탁했어요 (requested me to call)

**말하다 (to tell, say):**

- 조용히 하라고 말했어요 (told me to be quiet)

**제안하다 (to suggest):**

- 같이 가자고 제안했어요 (suggested going together)

### Softened Commands with -달라고

For softer requests, use -달라고 instead of -라고:

- 도와달라고 했어요 (asked me to help - softer)
- 와달라고 했어요 (asked me to come - softer)
- 전화하달라고 했어요 (asked me to call - softer)

This is more like "asked me to" rather than "told me to."

### Shortened Forms in Casual Speech

The shortened forms are very common:

- 빨리 가래 (telling me to go quickly)
- 먹지 말래 (telling me not to eat)
- 같이 가재 (suggesting to go together)
- 조용히 하래 (telling me to be quiet)

**Note**: Context is crucial with -래 since it can mean:
1. Statement: 간래 (says he's going)
2. Question: 가래 (asking if I'm going)
3. Command: 가래 (telling me to go)

### Commands with Objects and Complements

- 책을 읽으라고 했어요 (told me to read a book)
- 한국어로 말하라고 했어요 (told me to speak in Korean)
- 집에 빨리 오라고 했어요 (told me to come home quickly)
- 숙제를 내일까지 하라고 했어요 (told me to do homework by tomorrow)

### Multiple Commands

- 앉으라고 하고 기다리라고 했어요 (told me to sit and wait)
- 먹지 말라고 하고 운동하라고 했어요 (told me not to eat and to exercise)
