# Intermediate Korean

Welcome to the intermediate level! At this stage, you'll expand your ability to express complex ideas, report what others say, and use sophisticated grammatical structures.

## What You'll Learn

### Complex Grammar Patterns
Master advanced sentence structures that allow you to:

- Modify nouns with verb clauses (relative clauses)
- Convert verbs into nouns (nominalization)
- Express simultaneous actions and timing relationships
- Indicate purpose and intention clearly
- Describe results and changes
- Express degrees and extent

### Reported Speech
Learn to convey what others have said, asked, or commanded:

- Report statements in various tenses
- Report questions (both yes/no and wh-questions)
- Report commands and requests
- Use shortened colloquial forms

## Learning Path

1. **Complex Grammar** - Build sophisticated sentence structures
2. **Reported Speech** - Master indirect speech patterns
3. **Passive & Causative** - Understand voice changes
4. **Honorifics** - Navigate formal language properly
5. **Advanced Conjunctions** - Connect complex ideas

## Prerequisites

Before starting this level, you should be comfortable with:

- All basic grammar patterns
- Past, present, and future tenses
- Common verb forms and endings
- Basic particles and connectors
- Speech levels (informal and formal)

## Study Tips

**Practice Regularly**: These patterns require consistent exposure to internalize.

**Listen Actively**: Pay attention to how native speakers use these structures in dramas, podcasts, and conversations.

**Write Examples**: Create your own sentences using each pattern to reinforce learning.

**Mix Patterns**: Combine different grammar structures in single sentences for fluency.

## Ready to Begin?

Start with **Complex Grammar** to build your foundation in sophisticated sentence construction!
