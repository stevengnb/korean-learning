# Word Formation with Hanja

## Basic Principles

Korean Sino-Korean words are formed by combining hanja characters, each contributing meaning.

### Pattern: Character + Character = Word

**學 (study) + 校 (school) = 학교 (school)**
**生 (birth) + 日 (day) = 생일 (birthday)**

## Common Formation Patterns

### 1. Same Character Repeated

**人 + 人 = 인간 (人間) - human being**
**國 + 國 = 국가 (國家) - nation**
**時 + 時 = 시시 (時時) - constantly**

### 2. Opposite Concepts

**東 (east) + 西 (west) = 동서 (東西) - things, stuff**
**上 (up) + 下 (down) = 상하 (上下) - up and down, hierarchy**
**男 (man) + 女 (woman) = 남녀 (男女) - men and women**
**出 (exit) + 入 (enter) = 출입 (出入) - going in and out**

### 3. Related Concepts

**父 (father) + 母 (mother) = 부모 (父母) - parents**
**兄 (older brother) + 弟 (younger brother) = 형제 (兄弟) - brothers**
**姉 (older sister) + 妹 (younger sister) = 자매 (姉妹) - sisters**

### 4. Modifier + Main

**大 (big) + 學 (learning) = 대학 (大學) - university**
**小 (small) + 學 (learning) = 소학 (小學) - elementary learning**
**中 (middle) + 學 (learning) = 중학 (中學) - middle learning**
**高 (high) + 等 (class) = 고등 (高等) - high class**

### 5. Action + Object

**讀 (read) + 書 (book) = 독서 (讀書) - reading**
**食 (eat) + 事 (matter) = 식사 (食事) - meal**
**作 (make) + 品 (item) = 작품 (作品) - work of art**

## Understanding New Words

Once you know components, you can deduce meaning:

**未知的 word: 도서관 (圖書館)**
- 圖 (picture/plan) + 書 (book) + 館 (building)
- = Building for books = Library!

**未知的 word: 전화기 (電話機)**
- 電 (electricity) + 話 (speech) + 機 (machine)
- = Machine for electric speech = Telephone!

## Three-Character Words

**天氣圖 → 일기예보 (天氣豫報)**
- 天 (heaven) + 氣 (air) + 豫 (beforehand) + 報 (report)
- = Weather forecast

**自動車 → 자동차**
- 自 (self) + 動 (move) + 車 (vehicle)
- = Self-moving vehicle = Car

## Systematic Learning

### By Category

**Education:**
- 교육 (敎育) - education
- 교실 (敎室) - classroom
- 교사 (敎師) - teacher
- 교과서 (敎科書) - textbook

**Transportation:**
- 교통 (交通) - traffic/transportation
- 차량 (車輛) - vehicle
- 지하철 (地下鐵) - subway
- 비행기 (飛行機) - airplane

## Hanja Helps Distinguish Homophones

**사기:**
- 士氣 - morale
- 詐欺 - fraud
- 沙器 - earthenware
- 事記 - historical record

**Without hanja:** Confusing!
**With hanja:** Crystal clear!

## Modern Usage

**New words still formed with hanja:**
- 스마트폰 (smartphone) - 휴대폰 (携帶phone)
- 컴퓨터 (computer) - 전자계산기 (電子計算機)
- Hybrid: 인터넷 (internet) + 서점 (書店) = 인터넷 서점

## Practice

### Decode These Words

1. 남북 (南北)
2. 출발 (出發)
3. 한국어 (韓國語)
4. 외국인 (外國人)
5. 동서남북 (東西南北)

**Answers:**
1. South + North = North-South
2. Exit + Depart = Departure
3. Korea + Country + Language = Korean language
4. Outside + Country + Person = Foreigner
5. East + West + South + North = All directions

## Study Strategy

1. **Learn common characters first** (100-300)
2. **Group words by shared character**
3. **Notice patterns in formation**
4. **Deduce new word meanings**
5. **Verify with dictionary**

Hanja knowledge = vocabulary multiplication!
