# Complex Grammatical Patterns

## -는 셈이다 (Can be considered as, amounts to)

**Meaning:** Regarding something as, considering, amounts to

### Formation
- Action verbs: -는 셈이다
- Descriptive/Past: -(으)ㄴ 셈이다

### Usage

```
매일 운동하니까 돈을 절약하는 셈이에요.
Since I exercise daily, it's like saving money.

반값이니까 공짜나 마찬가지인 셈이다.
At half price, it's basically like it's free.

실패했지만 경험을 쌓은 셈이다.
Although I failed, it amounts to gaining experience.

10년을 기다렸으니 평생을 기다린 셈이다.
Having waited 10 years, it's like waiting a lifetime.
```

**Common patterns:**
- ~나 마찬가지인 셈이다 (it's as if...)
- ~(으)ㄴ 셈 치다 (consider it as...)

## -기 마련이다 (Bound to, naturally, it's natural that)

**Meaning:** It's natural/inevitable, bound to happen

### Formation
- All verbs: -기 마련이다

### Usage

```
사람은 실수하기 마련이다.
People are bound to make mistakes.

세월이 지나면 변하기 마련이에요.
As time passes, things are bound to change.

노력하면 성공하기 마련이다.
If you try hard, you're bound to succeed.

처음에는 어렵기 마련이에요.
It's naturally difficult at first.
```

**Expresses:**
- Natural law/principle
- Inevitable outcome
- Common sense
- Universal truth

## -는 법이다 (It's the way/rule that, it's how)

**Meaning:** That's how it is, that's the way, it's the rule

### Formation
- Action verbs: -는 법이다
- Descriptive: -(으)ㄴ 법이다

### Usage

```
노력하는 사람이 성공하는 법이다.
Those who try hard succeed - that's how it works.

시간이 지나면 잊는 법이에요.
As time passes, you forget - that's how it is.

부모는 자식을 사랑하는 법이다.
Parents love their children - that's the way it is.

급할수록 돌아가는 법이다.
The more urgent, the more you should take time - that's the rule.
```

**Similar to -기 마련이다 but emphasizes "the way things work"**

## -고 말다 (End up doing, finally do)

**Meaning:** Eventually, in the end, finally (often negative outcome)

### Formation
- All verbs: -고 말다

### Usage

```
결국 포기하고 말았어요.
In the end, I gave up.

비밀이 들키고 말았다.
The secret ended up being discovered.

결국 화를 내고 말았어요.
I ended up getting angry.

늦고 말았다.
I ended up being late.
```

**Often implies:**
- Regret
- Negative outcome
- Inevitability
- Finality

**Contrast:**
```
결국 성공했다. (neutral: finally succeeded)
결국 성공하고 말았다. (implies: against odds, or with mixed feelings)
```

## -아/어 내다 (Successfully complete, manage to)

**Meaning:** Complete action, succeed in doing, manage to do

### Formation
- All verbs: -아/어 내다

### Usage

```
어려운 시험을 통과해 냈어요.
I managed to pass the difficult exam.

끝까지 참아 냈다.
I endured to the end.

드디어 해냈어!
I finally did it!

힘들었지만 버텨 냈어요.
It was hard but I held out.
```

**Emphasizes:**
- Completion despite difficulty
- Achievement
- Persistence
- Success

## -기는 하다 (Do ~ but...)

**Meaning:** It's true that..., I do..., (admitting) but...

### Formation
- All verbs: -기는 하다

### Usage

```
비싸기는 한데 품질이 좋아요.
It is expensive, but the quality is good.

먹기는 먹었는데 별로였어요.
I did eat it, but it wasn't great.

알기는 아는데 설명하기는 어려워요.
I do know it, but it's hard to explain.

좋기는 좋은데 너무 비싸요.
It is good, but it's too expensive.
```

**Often followed by:** -ㄴ/는데, -지만 (contrastive)

## -ㄹ/을 뿐이다 (Only, merely, just)

**Meaning:** Only, merely, nothing but

### Formation
- All verbs: -ㄹ/을 뿐이다

### Usage

```
도와주고 싶을 뿐이에요.
I merely want to help.

최선을 다할 뿐이다.
I'll just do my best (nothing more).

모를 뿐이다.
I simply don't know.

기다릴 뿐이에요.
All I can do is wait.
```

**More formal than:** -만

**Often with negatives:**
```
알지 못할 뿐이다.
I simply don't know.
```

## -기만 하다 (Only do, do nothing but)

**Meaning:** Do nothing but, only keep doing

### Formation
- All verbs: -기만 하다

### Usage

```
놀기만 하고 공부를 안 해요.
(They) only play and don't study.

말만 하고 행동하지 않아요.
(They) only talk and don't act.

울기만 해요.
(They) do nothing but cry.
```

**Usually negative connotation** - doing too much of one thing

## Common Patterns Combined

These patterns often combine with others:

```
-기 마련이니까 (since it's bound to...)
-는 셈 치고 (considering it as...)
-고 말 거야 (will end up doing)
-아/어 내고 말았다 (ended up succeeding)
```

## Usage Tips

1. **-기 마련이다 / -는 법이다**: Use for general truths
2. **-는 셈이다**: Use for reframing/rationalizing
3. **-고 말다**: Use for finality (often negative)
4. **-아/어 내다**: Use for achievement despite difficulty
5. **-을 뿐이다**: Use for emphasis on limitation

These patterns appear frequently in:
- Formal writing
- News articles
- Academic texts
- Speeches
- Literary works

Master them for sophisticated Korean expression!
