# Academic Korean

## Essential Academic Vocabulary

### Research Terms
- 연구 (硏究) - research
- 논문 (論文) - paper, thesis
- 분석 (分析) - analysis
- 조사 (調査) - investigation, survey
- 실험 (實驗) - experiment
- 이론 (理論) - theory
- 가설 (假說) - hypothesis
- 방법론 (方法論) - methodology
- 자료 (資料) - data, materials
- 결과 (結果) - results
- 결론 (結論) - conclusion

### Academic Writing
- 서론 (序論) - introduction
- 본론 (本論) - body
- 문헌 검토 (文獻檢討) - literature review
- 참고문헌 (參考文獻) - references
- 인용 (引用) - citation, quotation
- 각주 (脚註) - footnote
- 목차 (目次) - table of contents

### Academic Expressions

**Introducing research:**
```
본 연구의 목적은 ~을/를 분석하는 것이다.
The purpose of this research is to analyze ~.

이 논문에서는 ~을/를 다룬다.
This paper deals with ~.
```

**Citing:**
```
김영수(2020)에 따르면...
According to Kim (2020)...

선행 연구에서는...
In previous research...
```

**Presenting results:**
```
연구 결과는 다음과 같다.
The research results are as follows.

분석 결과 ~으로 나타났다.
Analysis showed that ~.
```

**Concluding:**
```
결론적으로...
In conclusion...

본 연구의 시사점은...
The implications of this study are...
```

## Common Academic Patterns

### Formal statements
```
~라고 할 수 있다 - can be said that
~로 볼 수 있다 - can be seen as
~을/를 알 수 있다 - it can be known that
~라고 생각한다 - (I) think that
```

### Causation
```
~기 때문이다 - because
~으로 인해 - due to
~에 따라 - according to, following
~의 결과 - as a result of
```

This vocabulary is essential for Korean academic success!
