# Media & Journalism Korean

## News Vocabulary

### Media Terms
- 언론 (言論) - media, press
- 기자 (記者) - reporter, journalist
- 취재 (取材) - news gathering, coverage
- 보도 (報道) - news report
- 기사 (記事) - article
- 특종 (特種) - scoop, exclusive
- 속보 (速報) - breaking news
- 헤드라인 - headline

### Broadcasting
- 방송 (放送) - broadcast
- 뉴스 - news
- 앵커 - anchor
- 생방송 (生放送) - live broadcast
- 녹화 (錄畫) - recording
- 인터뷰 - interview

### News Expressions

**Reporting:**
```
~(이)라고 밝혔다 - stated that
~(이)라고 전했다 - reported that
~로 알려졌다 - it was learned that
~로 나타났다 - it appeared that
```

**Headlines:**
```
정부, 새 정책 발표
Government announces new policy

주가 급등
Stock prices surge
```

**Breaking news:**
```
속보입니다.
This is breaking news.

지금 들어온 소식입니다.
This is news just coming in.
```

Essential for understanding Korean news media!
