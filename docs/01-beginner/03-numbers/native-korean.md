## Numbers 1-10

### Basic Forms

| Number | Korean | When to Use |
|--------|--------|-------------|
| 1 | 하나 | Counting alone: "하나, 둘, 셋..." |
| 2 | 둘 | Counting alone |
| 3 | 셋 | Counting alone |
| 4 | 넷 | Counting alone |
| 5 | 다섯 | All situations (no change) |
| 6 | 여섯 | All situations (no change) |
| 7 | 일곱 | All situations (no change) |
| 8 | 여덟 | All situations (no change) |
| 9 | 아홉 | All situations (no change) |
| 10 | 열 | All situations (no change) |

### Modified Forms (Used with Counters)

The first four numbers change form when followed by a counter:

| Number | Standalone | With Counter | Example |
|--------|-----------|--------------|---------|
| 1 | 하나 | 한 | 한 개 (one thing) |
| 2 | 둘 | 두 | 두 명 (two people) |
| 3 | 셋 | 세 | 세 마리 (three animals) |
| 4 | 넷 | 네 | 네 살 (four years old) |

**Rule**: When a counter follows, use 한/두/세/네 instead of 하나/둘/셋/넷.

**Examples**:
- ✓ 사과 한 개 (one apple)
- ✗ 사과 하나 개 ❌
- ✓ 친구 두 명 (two friends)
- ✗ 친구 둘 명 ❌

### Exception: 하나, 둘, 셋, 넷 can stand alone
- 하나 주세요 (Give me one) - No counter, use full form
- 둘 더 필요해요 (I need two more) - No counter, use full form
- 한 번만 더 (Just one more time) - With counter 번, use shortened form

## Numbers 11-19

Form: **열** + (1-9)

| Number | Korean | Breakdown |
|--------|--------|-----------|
| 11 | 열하나 / 열한 | 열 + 하나 (standalone) / 열 + 한 (with counter) |
| 12 | 열둘 / 열두 | 열 + 둘 (standalone) / 열 + 두 (with counter) |
| 13 | 열셋 / 열세 | 열 + 셋 (standalone) / 열 + 세 (with counter) |
| 14 | 열넷 / 열네 | 열 + 넷 (standalone) / 열 + 네 (with counter) |
| 15 | 열다섯 | 열 + 다섯 |
| 16 | 열여섯 | 열 + 여섯 |
| 17 | 열일곱 | 열 + 일곱 |
| 18 | 열여덟 | 열 + 여덟 |
| 19 | 열아홉 | 열 + 아홉 |

**Examples**:
- 학생이 열한 명 있어요 (There are 11 students)
- 사탕 열두 개 주세요 (Give me 12 candies)
- 열다섯 시간 걸려요 (It takes 15 hours)

## Tens (20-90)

These numbers have unique names - they are NOT formed by multiplication like Sino-Korean:

| Number | Korean | Note |
|--------|--------|------|
| 20 | 스물 | Changes to 스무 before counters |
| 30 | 서른 | No change |
| 40 | 마흔 | No change |
| 50 | 쉰 | No change |
| 60 | 예순 | No change |
| 70 | 일흔 | No change |
| 80 | 여든 | No change |
| 90 | 아흔 | No change |

### Special Note: 스물 (20)
- Standalone or end of sentence: **스물**
  - 스물이에요 (It's 20)
  - 하나, 둘... 스물! (One, two... twenty!)
- Before counter: **스무**
  - 스무 살 (20 years old)
  - 스무 개 (20 things)
  - 스무 명 (20 people)

## Summary of Form Changes

### When to Use Modified Forms

**Pattern**: Number + Counter = Modified form (for 1-4 and 20)

| Original | Modified | Context |
|----------|----------|---------|
| 하나 | 한 | Before counter: 한 개, 한 명, 한 시 |
| 둘 | 두 | Before counter: 두 개, 두 명, 두 시 |
| 셋 | 세 | Before counter: 세 개, 세 명, 세 시 |
| 넷 | 네 | Before counter: 네 개, 네 명, 네 시 |
| 스물 | 스무 | Before counter: 스무 살, 스무 개 |

**No change for**: 5-10, 30-90 in all contexts

## Real-World Usage

### Ages (살)
- 저는 스물다섯 살이에요 (I'm 25 years old)
- 동생은 열세 살이에요 (My younger sibling is 13)
- 할머니는 일흔여덟 살이세요 (My grandmother is 78)

### Counting Things (개)
- 사과 다섯 개 (five apples)
- 연필 열두 개 (twelve pencils)
- 의자 서른 개 (thirty chairs)

### Counting People (명)
- 학생 스무 명 (20 students)
- 손님 일곱 명 (7 guests)
- 직원 마흔세 명 (43 employees)

### Hours (시)
- 한 시 (1 o'clock)
- 여섯 시 반 (6:30)
- 열한 시 (11 o'clock)

### Counting Animals (마리)
- 강아지 두 마리 (two puppies)
- 새 다섯 마리 (five birds)
- 물고기 스무 마리 (20 fish)

## Common Expressions

### Basic Counting
- 하나, 둘, 셋, 넷, 다섯! (One, two, three, four, five! - when counting)
- 열까지 세어 보세요 (Count to ten)

### Quantities
- 한 개만 주세요 (Just give me one)
- 두 개 더 주세요 (Give me two more)
- 몇 개 있어요? (How many are there?)

### Age Expressions
- 몇 살이에요? (How old are you?)
- 서른 살이 넘었어요 (I'm over 30)
- 아직 스무 살이에요 (I'm still 20)
