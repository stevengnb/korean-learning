# Present Tense

!!! abstract "Quick Reference"
    | Speech Level | Pattern | Example | Usage |
    |--------------|---------|---------|-------|
    | Informal Polite | Stem + 아/어요 | 가요, 먹어요 | Daily conversation |
    | Formal Polite | Stem + ㅂ/습니다 | 갑니다, 먹습니다 | Formal situations |
    | Informal | Stem + 아/어 | 가, 먹어 | Close friends, family |
    | Formal (Written) | Stem + ㄴ/는다 | 간다, 먹는다 | Books, narration |

## Present Tense Usage

Korean present tense is used for:
- Current actions or states
- Habitual actions
- General truths
- Future scheduled events (when context is clear)

## Informal Polite (해요체)

This is the most commonly used form in everyday polite conversation. Master this first.

### Formation Rules

**Step 1:** Remove 다 from dictionary form
**Step 2:** Apply vowel harmony (아 for bright vowels ㅏ/ㅗ, 어 for others)
**Step 3:** Add 요 for politeness

### Regular Verb Conjugation Table

| Dictionary Form | Stem | Last Vowel | Ending | Conjugation | Translation |
|----------------|------|------------|--------|-------------|-------------|
| 가다 | 가 | ㅏ (bright) | 아요 | 가요 | go |
| 오다 | 오 | ㅗ (bright) | 아요 | 와요 | come |
| 보다 | 보 | ㅗ (bright) | 아요 | 봐요 | see/watch |
| 먹다 | 먹 | ㅓ (dark) | 어요 | 먹어요 | eat |
| 서다 | 서 | ㅓ (dark) | 어요 | 서요 | stand |
| 자다 | 자 | ㅏ (bright) | 아요 | 자요 | sleep |
| 읽다 | 읽 | ㅣ (dark) | 어요 | 읽어요 | read |
| 쉬다 | 쉬 | ㅣ (dark) | 어요 | 쉬어요 | rest |
| 배우다 | 배우 | ㅜ (dark) | 어요 | 배워요 | learn |
| 주다 | 주 | ㅜ (dark) | 어요 | 줘요 | give |

### 하다 Verbs

All verbs ending in 하다 follow the same pattern: 하 + 어 → 해

| Dictionary Form | Conjugation | Translation |
|----------------|-------------|-------------|
| 하다 | 해요 | do |
| 공부하다 | 공부해요 | study |
| 일하다 | 일해요 | work |
| 운동하다 | 운동해요 | exercise |
| 요리하다 | 요리해요 | cook |
| 청소하다 | 청소해요 | clean |
| 사랑하다 | 사랑해요 | love |
| 생각하다 | 생각해요 | think |

## Formal Polite (합니다체)

Used in formal situations, business, presentations, and news.

### Formation Rules

- **Stems ending in vowel:** Add -ㅂ니다
- **Stems ending in consonant:** Add -습니다

| Dictionary Form | Stem Type | Conjugation | Translation |
|----------------|-----------|-------------|-------------|
| 가다 | Vowel (가) | 갑니다 | go |
| 오다 | Vowel (오) | 옵니다 | come |
| 하다 | Vowel (하) | 합니다 | do |
| 먹다 | Consonant (먹) | 먹습니다 | eat |
| 읽다 | Consonant (읽) | 읽습니다 | read |
| 만들다 | Consonant (만들) | 만듭니다 | make |
| 듣다 | Consonant (듣) | 듣습니다 | listen |
| 앉다 | Consonant (앉) | 앉습니다 | sit |

## Irregular Verbs in Present Tense

### ㄷ Irregular

When ㄷ-final stems are followed by a vowel, ㄷ changes to ㄹ.

| Dictionary Form | Regular Would Be | Actual | Translation |
|----------------|------------------|--------|-------------|
| 듣다 | 듣어요 ✗ | 들어요 ✓ | listen |
| 걷다 | 걷어요 ✗ | 걸어요 ✓ | walk |
| 묻다 | 묻어요 ✗ | 물어요 ✓ | ask |
| 싣다 | 싣어요 ✗ | 실어요 ✓ | load |

**Exception:** 믿다 (to believe) is regular → 믿어요

### ㅂ Irregular

When ㅂ-final stems are followed by a vowel, ㅂ changes to 우.

| Dictionary Form | Regular Would Be | Actual | Translation |
|----------------|------------------|--------|-------------|
| 춥다 | 춥어요 ✗ | 추워요 ✓ | be cold |
| 덥다 | 덥어요 ✗ | 더워요 ✓ | be hot |
| 어렵다 | 어렵어요 ✗ | 어려워요 ✓ | be difficult |
| 쉽다 | 쉽어요 ✗ | 쉬워요 ✓ | be easy |
| 돕다 | 돕어요 ✗ | 도와요 ✓ | help |
| 아름답다 | 아름답어요 ✗ | 아름다워요 ✓ | be beautiful |

**Exception:** 좁다 (to be narrow) is regular → 좁아요

### ㅡ Irregular

When stems ending in ㅡ are followed by 아/어, the ㅡ is dropped.

| Dictionary Form | Process | Conjugation | Translation |
|----------------|---------|-------------|-------------|
| 쓰다 | 쓰 + 어 → ㅡ drops | 써요 | write/use |
| 끄다 | 끄 + 어 → ㅡ drops | 꺼요 | turn off |
| 크다 | 크 + 어 → ㅡ drops | 커요 | be big |
| 예쁘다 | 예쁘 + 어 → ㅡ drops | 예뻐요 | be pretty |
| 슬프다 | 슬프 + 어 → ㅡ drops | 슬퍼요 | be sad |
| 기쁘다 | 기쁘 + 어 → ㅡ drops | 기뻐요 | be happy |

**Note:** If there's ㅏ or ㅗ before ㅡ, use 아 instead of 어: 나쁘다 → 나빠요 (be bad)

### ㄹ Irregular

ㄹ drops before ㄴ, ㅂ, or ㅅ, but this mainly affects formal polite form in present tense.

| Dictionary Form | Informal Polite | Formal Polite | Translation |
|----------------|----------------|---------------|-------------|
| 살다 | 살아요 | 삽니다 (ㄹ drops) | live |
| 만들다 | 만들어요 | 만듭니다 (ㄹ drops) | make |
| 알다 | 알아요 | 압니다 (ㄹ drops) | know |
| 팔다 | 팔아요 | 팝니다 (ㄹ drops) | sell |

### ㅅ Irregular

ㅅ drops when followed by a vowel.

| Dictionary Form | Process | Conjugation | Translation |
|----------------|---------|-------------|-------------|
| 짓다 | 짓 → 지 + 어 | 지어요 | build |
| 낫다 | 낫 → 나 + 아 | 나아요 | be better |

## Examples with Context

### Daily Routines

1. 저는 매일 아침 7시에 일어나요. (I wake up at 7 AM every morning.)
2. 아침을 먹어요. (I eat breakfast.)
3. 학교에 가요. (I go to school.)
4. 수업을 들어요. (I attend class.)
5. 점심에 친구들을 만나요. (I meet friends at lunch.)
6. 도서관에서 공부해요. (I study in the library.)
7. 저녁에 집에 와요. (I come home in the evening.)
8. 가족과 저녁을 먹어요. (I eat dinner with family.)
9. TV를 봐요. (I watch TV.)
10. 밤 11시에 자요. (I sleep at 11 PM.)

### States and Descriptions

1. 날씨가 추워요. (The weather is cold.)
2. 이 음식이 매워요. (This food is spicy.)
3. 한국어가 어려워요. (Korean is difficult.)
4. 이 책이 재미있어요. (This book is interesting.)
5. 제 방이 커요. (My room is big.)
6. 그 사람이 예뻐요. (That person is pretty.)

### Habitual Actions

1. 저는 커피를 좋아해요. (I like coffee.)
2. 주말에 운동해요. (I exercise on weekends.)
3. 한국 드라마를 자주 봐요. (I often watch Korean dramas.)
4. 매일 한국어를 공부해요. (I study Korean every day.)

## Practice

2. Conjugate to informal polite: 춥다, 돕다, 아름답다
3. Conjugate to informal polite: 쓰다, 크다, 예쁘다
4. Conjugate to formal polite: 살다, 만들다, 알다
5. Translate to Korean: "I eat breakfast every morning."
6. Translate to Korean: "Korean is difficult but interesting."
7. Conjugate and explain the irregularity: 걷다 → ?
8. Choose correct form: 날씨가 (덥어요 / 더워요).
9. Create a sentence using: 공부하다 (present tense)
10. Conjugate both forms: 듣다 → informal polite ?, formal polite ?

1. Informal polite conjugations (ㄷ irregular):
   - 걷다 → 걸어요 (ㄷ → ㄹ)
   - 듣다 → 들어요 (ㄷ → ㄹ)
   - 묻다 → 물어요 (ㄷ → ㄹ)

2. Informal polite conjugations (ㅂ irregular):
   - 춥다 → 추워요 (ㅂ → 우)
   - 돕다 → 도와요 (ㅂ → 우 + ㅏ → 와)
   - 아름답다 → 아름다워요 (ㅂ → 우)

3. Informal polite conjugations (ㅡ irregular):
   - 쓰다 → 써요 (ㅡ drops)
   - 크다 → 커요 (ㅡ drops)
   - 예쁘다 → 예뻐요 (ㅡ drops)

4. Formal polite conjugations (ㄹ irregular):
   - 살다 → 삽니다 (ㄹ drops before ㅂ)
   - 만들다 → 만듭니다 (ㄹ drops before ㅂ)
   - 알다 → 압니다 (ㄹ drops before ㅂ)

5. 저는 매일 아침 아침을 먹어요. / 저는 매일 아침에 아침을 먹어요.

6. 한국어는 어렵지만 재미있어요.
   (Or: 한국어가 어려워요. 그렇지만 재미있어요.)

7. 걷다 → 걸어요
   - Stem: 걷
   - This is a ㄷ irregular verb
   - When ㄷ meets a vowel, it changes to ㄹ
   - 걷 + 어 → 걸 + 어 → 걸어요

8. 날씨가 더워요. (Correct - ㅂ irregular: 덥다 → 더워요)

9. Sample answer: 저는 매일 한국어를 공부해요. (I study Korean every day.)
   Or: 도서관에서 공부해요. (I study in the library.)

10. 듣다 conjugations:
    - Informal polite: 들어요 (ㄷ → ㄹ before vowel)
    - Formal polite: 듣습니다 (no vowel follows, so ㄷ stays)
