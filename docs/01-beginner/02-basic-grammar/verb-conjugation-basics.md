# Verb Conjugation Basics

!!! abstract "Quick Reference"
    | Component | Description | Example |
    |-----------|-------------|---------|
    | Verb Stem | Remove 다 from dictionary form | 가다 → 가 |
    | Vowel Harmony | ㅏ/ㅗ → 아, Others → 어 | 가 → 가 + 아 |
    | Formal Polite | Add 요 to informal polite | 가요 → 가요 |
    | Informal Polite | 아/어 ending | 가 → 가 + 아 → 가 |

## Understanding Korean Verb Structure

Korean verbs in their dictionary form always end in 다. To conjugate a verb, you must first identify the **verb stem** by removing 다.

### Step 1: Finding the Verb Stem

| Dictionary Form | Remove 다 | Verb Stem |
|----------------|-----------|-----------|
| 가다 (to go) | 가~~다~~ | 가 |
| 먹다 (to eat) | 먹~~다~~ | 먹 |
| 보다 (to see) | 보~~다~~ | 보 |
| 공부하다 (to study) | 공부하~~다~~ | 공부하 |
| 마시다 (to drink) | 마시~~다~~ | 마시 |

### Step 2: Understanding Vowel Harmony

Korean uses **vowel harmony** to determine which ending to attach. This creates a more natural sound flow.

**Rule:** Look at the last vowel in the verb stem:

- **Bright vowels** (ㅏ, ㅗ) → Use **아** endings
- **Dark vowels** (all others) → Use **어** endings

| Verb Stem | Last Vowel | Harmony Type | Ending |
|-----------|------------|--------------|--------|
| 가 | ㅏ | Bright | 아 |
| 오 | ㅗ | Bright | 아 |
| 먹 | ㅓ | Dark | 어 |
| 보 | ㅗ | Bright | 아 |
| 마시 | ㅣ | Dark | 어 |
| 배우 | ㅜ | Dark | 어 |

### Step 3: Combining Stem and Ending

When the stem ends in the same vowel as the ending, they contract:

| Stem | + Ending | = Result |
|------|----------|----------|
| 가 | + 아 | = 가 (가 + 아 contracts) |
| 오 | + 아 | = 와 (오 + 아 contracts) |
| 서 | + 어 | = 서 (서 + 어 contracts) |
| 보 | + 아 | = 봐 (보 + 아 contracts) |

**When no contraction occurs:**

| Stem | + Ending | = Result |
|------|----------|----------|
| 먹 | + 어 | = 먹어 |
| 살 | + 아 | = 살아 |
| 마시 | + 어 | = 마셔 (ㅣ + 어 → ㅕ) |

### Step 4: Formality Levels

Korean has different speech levels. The most common for learners:

**Informal Polite (해요체):**
- Used in everyday polite conversation
- Formation: Stem + 아/어요

**Formal Polite (합니다체):**
- Used in formal situations, presentations, news
- Formation: Stem + ㅂ니다/습니다

| Verb | Informal Polite | Formal Polite |
|------|-----------------|---------------|
| 가다 | 가요 | 갑니다 |
| 먹다 | 먹어요 | 먹습니다 |
| 보다 | 봐요 | 봅니다 |
| 하다 | 해요 | 합니다 |

### 하다 Verbs: Special Case

Verbs ending in 하다 have a special contraction:

- 하 + 어 → **해**
- 공부하다 → 공부해요
- 운동하다 → 운동해요
- 요리하다 → 요리해요

## Detailed Conjugation Process

### Example 1: 가다 (to go)

1. Remove 다: 가~~다~~ → **가**
2. Check last vowel: ㅏ (bright) → use **아**
3. Combine: 가 + 아 → **가** (contraction)
4. Add 요: 가 + 요 → **가요**

### Example 2: 먹다 (to eat)

1. Remove 다: 먹~~다~~ → **먹**
2. Check last vowel: ㅓ (dark) → use **어**
3. Combine: 먹 + 어 → **먹어**
4. Add 요: 먹어 + 요 → **먹어요**

### Example 3: 만들다 (to make)

1. Remove 다: 만들~~다~~ → **만들**
2. Check last vowel: ㅡ (dark) → use **어**
3. Combine: 만들 + 어 → **만들어** (ㅡ drops before 어)
4. Add 요: 만들어 + 요 → **만들어요**

## Common Contraction Patterns

| Pattern | Example | Result |
|---------|---------|--------|
| ㅏ + 아 | 가 + 아 | 가 |
| ㅗ + 아 | 오 + 아 | 와 |
| ㅓ + 어 | 서 + 어 | 서 |
| ㅜ + 어 | 주 + 어 | 줘 |
| ㅣ + 어 | 마시 + 어 | 마셔 |
| 하 + 어 | 하 + 어 | 해 |
| ㅡ + 어 | 쓰 + 어 | 써 (ㅡ drops) |

## Examples with Translations

1. 저는 학교에 가요. (I go to school.)
2. 밥을 먹어요. (I eat rice/food.)
3. 친구를 봐요. (I see a friend.)
4. 한국어를 공부해요. (I study Korean.)
5. 물을 마셔요. (I drink water.)
6. 집에서 쉬어요. (I rest at home.)
7. 음악을 들어요. (I listen to music.)
8. 편지를 써요. (I write a letter.)
9. 운동을 해요. (I exercise.)
10. 책을 읽어요. (I read a book.)

## Irregular Verbs

Some verbs follow special conjugation rules. These will be covered in detail in each tense section, but here's a preview:

### ㄷ Irregular
- 듣다 (to listen) → 들어요 (ㄷ → ㄹ before 어)
- 걷다 (to walk) → 걸어요

### ㅂ Irregular
- 춥다 (to be cold) → 추워요 (ㅂ → 우 before 어)
- 어렵다 (to be difficult) → 어려워요

### ㅡ Irregular
- 쓰다 (to write) → 써요 (ㅡ drops before 어)
- 끄다 (to turn off) → 꺼요

### ㄹ Irregular
- 살다 (to live) → 살아요 (ㄹ drops before ㄴ, ㅂ, ㅅ)
- 만들다 (to make) → 만들어요
