# Beginner Level

Welcome to the beginner level of your Korean learning journey. This section covers the absolute fundamentals you need to start communicating in Korean.

## What You'll Learn

| Section | Topics | Goal |
|---------|--------|------|
| 1.1 Hangul | Letters, syllables, pronunciation | Read and write Korean script |
| 1.2 Basic Grammar | Sentence structure, particles, tenses | Form basic sentences |
| 1.3 Numbers | Native & Sino-Korean systems, counters | Count and tell time |
| 1.4 Basic Expressions | Greetings, introductions, questions | Survive basic interactions |
| 1.5 Basic Vocabulary | Family, food, places, common verbs | Build foundation vocabulary |

## Prerequisites

None. This level assumes zero Korean knowledge.

## Time Estimate

With consistent daily study (30-60 minutes):

- Hangul mastery: 1-2 weeks
- Basic grammar foundation: 4-6 weeks
- Complete beginner level: 2-3 months

## Tips for Success

!!! tip "Hangul First"
    Do not skip or rush Hangul. A solid foundation here makes everything else easier.

!!! tip "Active Practice"
    Don't just read. Write out every example. Say sentences out loud.

!!! tip "Use Anki"
    Since you're already using Anki, add new vocabulary as you learn it. Include example sentences, not just isolated words.
