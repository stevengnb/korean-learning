## Size and Dimension

| Korean | Meaning | Present Form | Example |
|--------|---------|--------------|---------|
| 크다 | to be big | 커요 | 집이 커요 (The house is big) |
| 작다 | to be small | 작아요 | 방이 작아요 (The room is small) |
| 길다 | to be long | 길어요 | 머리가 길어요 (Hair is long) |
| 짧다 | to be short | 짧아요 | 치마가 짧아요 (The skirt is short) |
| 넓다 | to be wide/spacious | 넓어요 | 거실이 넓어요 (The living room is spacious) |
| 좁다 | to be narrow | 좁아요 | 길이 좁아요 (The road is narrow) |
| 높다 | to be high/tall | 높아요 | 산이 높아요 (The mountain is high) |
| 낮다 | to be low | 낮아요 | 목소리가 낮아요 (The voice is low) |
| 두껍다 | to be thick | 두꺼워요 | 책이 두꺼워요 (The book is thick) |
| 얇다 | to be thin | 얇아요 | 종이가 얇아요 (The paper is thin) |

**Usage Examples:**
- 이 가방이 너무 커요. (This bag is too big.)
- 손이 작아요. (My hands are small.)
- 이 바지가 너무 길어요. (These pants are too long.)
- 아파트가 넓고 좋아요. (The apartment is spacious and nice.)

## Important Usage Notes

### Adjectives as Predicates

Korean adjectives function as verbs and don't need a copula:
- 날씨가 좋다. (The weather is good.) - NOT 좋이다
- 꽃이 예쁘다. (The flower is pretty.)

### Adjectives Modifying Nouns

When modifying nouns, adjectives take different forms:
- Present: -(으)ㄴ
  - 큰 집 (big house)
  - 작은 방 (small room)
  - 예쁜 꽃 (pretty flower)
  - 좋은 사람 (good person)

### Irregular Adjectives

Common irregular patterns:
- ㅂ irregular: 춥다 → 추워요, 덥다 → 더워요
- ㄷ irregular: 듣다 → 들어요
- ㅎ irregular: 빨갛다 → 빨개요
- ㅡ irregular: 크다 → 커요, 쓰다 → 써요

### Negative Form

To make negative:
- 안 + adjective: 안 좋아요 (not good)
- Adjective + 지 않다: 좋지 않아요 (not good)

**Examples:**
- 안 예뻐요. (It's not pretty.)
- 어렵지 않아요. (It's not difficult.)
